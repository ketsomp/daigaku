## Responsive Behavior

### Window Resize

- Main window: 1000x700 minimum
- Login window: Fixed 400x300
- Tables: Stretch to fit width
- Buttons: Fixed height (40-60px)

### Font Sizes

- Title: 18-20pt, Bold
- Headers: 14-16pt, Bold
- Body text: 10-12pt
- Expression (flashcard): 32pt, Bold
- Definition: 16pt
- Button text: 12-16pt

## Keyboard Shortcuts

- F5: Refresh
- Ctrl+Q: Quit
- Enter: Show Answer (flashcard mode)
- 1-5: Rate answer (flashcard mode)
- Esc: Cancel/Close dialog

## Visual States

### Button States

1. Normal: Base color
2. Hover: Slightly darker (opacity 0.8)
3. Pressed: Much darker
4. Disabled: Gray, 50% opacity

### Card States

1. Question: Show expression only
2. Answer: Show definition + example
3. Correct: Brief green flash
4. Incorrect: Brief red flash

## Accessibility

- High contrast text
- Clear focus indicators
- Large clickable areas (min 40x40px)
- Descriptive button labels
- Alt text for icons
